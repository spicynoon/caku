Selamat datang di Caku - Catatan Kuliah, solusi terdepan untuk membuat pengalaman kuliah kamu lebih teratur dan produktif. Caku didesain khusus untuk membantu kamu mengelola tugas dan catatan perkuliahan dengan mudah dan efisien. Aplikasi Caku sangat bermanfaat bagi pelajar terkhusus mahasiswa yang ingin meningkatkan produktivitas dan efisiensi dalam proses akademik. Caku memudahkan penggunanya untuk memanage tugas dan catatan selama perkuliahan sehingga catatan dan tugas dapat dipantau di satu tempat. Sebagai gambaran, kamu dapat melihat [Demo aplikasi Caku 📽️di sini](URL_demo_aplikasi), atau kamu dapat melihat detail gambaran umum caku di bawah ini:

<details open>
  <summary><b>📜Fitur Utama</b></summary>
  
1. Manajemen Tugas
    - Kamu dapat dengan mudah mengatur dan mencatat tugas apa saja yang diberikan oleh dosenmu.
    - Pengelompokkan tugas berdasarkan mata kuliah memudahkan kamu untuk mengetahui tugas yang harus diselesaikan berdasarkan prioritas.
2.  Catatan Kuliah
    - Buat catatan perkuliahan dengan rapi dan teratur!
    - Catatan dikelompokkan berdasarkan mata kuliah yang memudahkan kamu dalam menyusul dan mengedit catatan dengan mudah.
</details>

  <summary><h3>📲Melihat Folder Tugas dan Catatan</h3>
  </summary>

<details open>
  <summary><h3>📲Manajemen Tugas</h3>
  </summary>    
  Jika dosenmu memberi tugas baru, kamu dapat menekan folder mata kuliah terkait dan menambahkan tugas sebagaimana <code>gambar kedua</code>. Kemudian, jika folder mata kuliah belum ada, kamu dapat dengan mudah membuatnya sebagaimana <code>gambar pertama</code>. List tugas di mata kuliah tertentu akan terlihat setelah kamu menekan folder mata kuliah sebagiamana <code>gambar ketiga</code><p>

<details open>
  <summary><h3>📲Catatan Kuliah</h3>
  </summary>    
  Sama halnya dengan tugas, kamu dapat membuat catatan berdasarkan mata kuliah terkait. Hal ini dapat memudahkanmu dalam membuat catatan baru sehingga catatan tiap mata kuliah tidak tercampur. Alur dari pembuatan catatan dan folder catatan kurang lebih sama dengan saat mengatur tugas, hal ini sangat memudahkan kamu sebagai pengguna baru dalam mengenali dan menggunakan Caku.<p>
